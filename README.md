## 2️ rule_based_chatbot.py  
This script is a simple **rule-based chatbot** built using Python.  
It replies to the user based on predefined conditions (if–elif–else).

###Features
- Responds to greetings (hello, hi, hey)
- Answers "how are you"
- Responds to farewells (bye, see you)
- Tells its name when asked
- Provides help message
- Gives random fallback responses if input is not understood
- Loop continues until the user types **quit**

### How to Run
1. Open terminal in the folder where the file is saved  
2. Run this command:python chatbot.py

# 3 Stock Portfolio Tracker

A simple command-line Python tool for tracking a small stock portfolio.  
Users enter stock symbols and quantities, and the program calculates each stock’s value and total investment.  
You can optionally save the portfolio summary to a text file.


##  Features

- Hardcoded stock price dictionary (easy to modify)
- Interactive user input for:
  - Stock symbol  
  - Number of shares  
- Validates user input
- Calculates:
  - Per-stock value
  - Total investment amount
- Option to save the summary to portfolio_summary.txt


##  Requirements

- Python 3.x  
(No external libraries are required.)
## ▶️ How to Run
1. Open a terminal and run: stock_tracker.py
